# Restaurant Management System (PHP & MySQL)

## Introduction
The Restaurant Management System is a database-driven application built using PHP and MySQL.  
It is designed to manage daily restaurant operations such as menu management, table reservations, orders, customers, staff roles, and inventory tracking.

This project helps restaurant administrators and staff efficiently handle restaurant workflows in a structured and organized way.

---

## Features
- User management with roles (Admin, Manager, Staff)
- Menu categories and menu items management
- Table management with capacity and status
- Customer information management
- Table reservations system
- Order and order items management
- Inventory management with stock tracking
- Inventory transaction history (in/out)
- Secure database structure with foreign key relationships

## Database Structure
The database includes the following main tables:
- users
- categories
- menu_items
- tables
- customers
- reservations
- orders
- order_items
- inventory
- inventory_transactions

All tables are properly connected using foreign keys to maintain data integrity.
